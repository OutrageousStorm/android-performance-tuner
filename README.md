# ⚡ Android Performance Tuner

Scripts to optimize Android device performance — disable unnecessary animations, boost RAM, optimize storage, limit background bloat.

## Scripts

| Script | What it does |
|--------|-------------|
| `performance_mode.sh` | One-shot performance tuning |
| `animations.sh` | Disable/enable animations |
| `memory_boost.sh` | Clear RAM, disable background apps |
| `io_optimize.sh` | Change I/O scheduler to deadline |
| `battery_gaming.sh` | Gaming profile — max performance, max refresh rate |
| `monitor.py` | Real-time performance stats |
